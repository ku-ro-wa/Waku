# ML-Powered Career Recommender

A web-based app that recommends career paths based on a user's resume or answers to a short quiz.

## Features

- Resume and quiz input options
- Personality and skills-based matching
- Career recommendations with descriptions

## MVP Goals

- Input parsing (quiz or resume)
- Rule-based or ML-based matching
- Clean, interactive interface

## Tech Stack

- Python
- [Flask/Streamlit]
- scikit-learn / spaCy / sentence-transformers

## Future Plans

- Add career fit scores
- Include skills gap analysis and upskilling suggestions
